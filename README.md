# 逃离沙漠

一个横屏手机网页小游戏，包含小企鹅逃离沙漠、技能球、星星、10000 米终点动画，以及分数和距离排行榜。

## 本地运行

```bash
npm start
```

打开：

```text
http://127.0.0.1:4173/index.html
```

## 需要上传的文件

- `index.html`
- `api/leaderboard.js`
- `server.js`
- `package.json`
- `leaderboard.json`
- `.gitignore`

## Supabase 排行榜数据库

在 Supabase 的 SQL Editor 里执行：

```sql
create table if not exists leaderboard_entries (
  id bigint generated by default as identity primary key,
  name text not null,
  score integer not null default 0,
  distance integer not null default 0,
  created_at timestamptz not null default now()
);

create index if not exists leaderboard_entries_score_idx
on leaderboard_entries (score desc);

create index if not exists leaderboard_entries_distance_idx
on leaderboard_entries (distance desc);
```

上线到 Vercel 时，在 Environment Variables 里添加：

- `SUPABASE_URL`
- `SUPABASE_SERVICE_ROLE_KEY`

Vercel 版本需要设置这两个变量，排行榜才会写入 Supabase。`server.js` 本地运行时如果没有设置这两个变量，会自动使用本地的 `leaderboard.json` 作为备用排行榜。

## Vercel 永久在线

1. 在 GitHub 新建仓库，把上面的文件上传。
2. 打开 Vercel，选择 `Add New` -> `Project`。
3. 连接刚才的 GitHub 仓库。
4. 设置：
   - Framework Preset: `Other`
   - Root Directory: 留空
   - Build Command: 留空或使用默认值
   - Output Directory: 留空
5. 在 Environment Variables 里添加：
   - `SUPABASE_URL`
   - `SUPABASE_SERVICE_ROLE_KEY`
6. 部署完成后，把 Vercel 给出的 `https://...vercel.app` 链接发给朋友即可。

注意：Vercel 版本的排行榜依赖 Supabase 环境变量。不要把 `SUPABASE_SERVICE_ROLE_KEY` 写进 `index.html`，也不要公开分享。
